# Apprendre l'IA avec Claude

Guide pas à pas, en français, pour apprendre l'IA en partant de zéro, avec un focus sur Claude et Claude Desktop.

Public visé : une personne à l'aise avec l'informatique (a déjà vu du code) mais sans aucune connaissance en IA.

## Contenu

Une seule page HTML autonome ([index.html](index.html)) : styles et diagrammes SVG inclus, aucune dépendance externe, lisible hors ligne.

1. C'est quoi, une IA ? Comprendre les grands modèles de langage
2. Claude et Anthropic : qui fait quoi ?
3. Premiers pas : compte, installation, première conversation
4. Bien parler à Claude : les bases du prompt
5. Claude Desktop en mode avancé : projets, connecteurs, artifacts
6. Agents et skills : quand Claude passe à l'action
7. Limites et bons réflexes
8. Aller plus loin

Plus un glossaire de 12 termes.
